Install Docker Desktop
Sign in to Dockerhub and Docker Desktop
Docker Desktop:
    > Check docker install on cli: "docker"
    > Pull the hello-world image and run it.

Build docker image: docker build -t mlops-docker-demo .
Run the container: docker run -p 5000:5000 mlops-docker-demo
Tag your image: docker tag mlops-docker-demo adeelajmal2468/mlops-docker-demo:latest1
Assure that you are logged in: docker login
Push image to dockerhub: docker push adeelajmal2468/mlops-docker-demo:latest1
Pull image from dockerhub: docker pull  adeelajmal2468/mlops-docker-demo:latest1
Run the Pulled image: docker run -p 5000:5000 adeelajmal2468/mlops-docker-demo:latest1
